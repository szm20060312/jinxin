### 关键决策

- EdgeOne 原项目 jinxin 为 GitHub 绑定类型（Provider: Github），CLI deploy 只能新建 Upload 项目。正确方式是 push GitHub → 触发 EdgeOne 自动部署。
- git 推送通过系统代理 `127.0.0.1:7890` + `gh auth token` OAuth2 认证解决。
- 新手教程改为每次新开局弹出，不再持久化 localStorage 标记，便于演示和评审。
