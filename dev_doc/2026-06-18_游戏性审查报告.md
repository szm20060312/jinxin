# 《晋·信》游戏性审查报告

> **日期**：2026年6月18日  
> **角色**：游戏策划  
> **范围**：核心循环、策略深度、叙事整合、终局体验、复玩价值  
> **方法**：模拟典型玩家 30 分钟 session，追踪 S1→S20 完整心流曲线

---

## 一、核心循环：无进化的机械重复

### 1.1 内容衰减曲线

```
S1:  晨间简报(4卡) → 季度决策(6项) → 模拟 → [新鲜感峰值]
S3:  晨间简报(4卡) → 季度决策(6项) → 模拟 → [已见所有模板]
S8:  晨间简报(4卡) → 季度决策(6项) → 模拟 → [开始无脑点击]
S16: 晨间简报(4卡) → 季度决策(6项) → 模拟 → [机械劳动]
```

**根因**：玩家在第 3 季已经见过所有决策类型、所有简报模板、所有事件分类。此后 17 季不再出现任何新内容。没有机制解锁、没有新挑战类型、没有复杂度爬升。

### 1.2 决策疲劳

每季 **6~7 项决策 × 每项 2~4 选项**：

| 统计 | 数值 |
|------|------|
| 每季选择次数 | 6~7 次 |
| 每季选项阅读量 | ~18 个选项标签 |
| 每年决策总量 | 24~28 次 |
| 5 年通关总选择量 | **120~140 次** |

但其中 **70% 的选项效果在 ±1~3 信誉之间**（×3 放大后），玩家很快感知到"选哪个都差不多"，决策退化为仪式性点击。

### 1.3 晨间简报：从"情报"到"障碍"

简报的设计意图是"让玩家在决策前获取信息"。但实际体验是：

- 简报内容是**纯信息**，玩家只能阅读，不能操作
- 必须逐条点击"知晓"才能进入决策
- 第 3 季起，玩家开始**无脑跳过**——简报失去了信息价值
- 简报的 `reliability`（高/中/低）标记在机械点击中完全被忽略

## 二、策略深度：没有"建设"的经营游戏

### 2.1 缺失的建设维度

玩家经营 5 年票号，**不能**：

| 不可做 | 应有的体验 |
|--------|-----------|
| 开第三家分号 | 从 2 分号扩展到 5 分号网络 |
| 升级分号等级 | 二等→一等，业务量提升 |
| 建造库房/设驿站 | 降低在途风险、加速白银流转 |
| 培养掌柜 | 送掌柜去别号学习、提升能力 |
| 投资基础设施 | 修路、设镖局、建仓库 |

所有"发展"只是数字涨跌。玩家感受不到"我在建造什么"。

### 2.2 年度规划的虚假选择

年终 Step 4 的三种来年方向：

| 选项 | 标签 | 实际效果 |
|------|------|---------|
| 开疆拓土 | "进取" | 无机械效果 |
| 固本培元 | "稳健" | 无机械效果 |
| 信誉至上 | "保守" | 无机械效果 |

玩家选什么都一样。这个步骤只提供 RP 价值，没有策略意义。

### 2.3 没有中期目标

从 S1 到结局（最短 5 年），玩家在等待一个遥远的终点，中间没有任何里程碑：

- 没有"第一笔万两大单"的阶段性成就
- 没有"首次获得官府嘉奖"的荣誉时刻
- 没有"分号扭亏为盈"的转折点
- 没有年份回顾（"今年比去年..."）

5 年是一段漫长的空白，**没有值得纪念的瞬间**。

### 2.4 人脉系统形同虚设

官府 / 商帮 / 东家三个数值会涨跌，但：

- 除了作为事件门槛，**没有任何主动使用方式**
- 不能"消耗 20 点官府好感换取一个特权"
- 不能"用人脉解锁新的商业机会"
- 玩家从不主动查看人脉面板

它们是三个"存在但无用"的数字。

## 三、叙事与机制：两个平行宇宙

### 3.1 故事事件像"弹出广告"

5 个主线事件的体验：

```
正常经营 → [突然跳出] 事件对话框 → 读描述 → 二选一 → 消失 → 继续经营
```

处理时间 < 20 秒。事件与经营循环没有互动：

- 官银汇兑选择后，后续季度官府关系没有差异化反馈
- 汉口洪灾选择后，汉口分号经营状态没有实际变化
- 事件之间没有因果链（"上次你帮了官府，所以这次..."）

### 3.2 突发事件缺乏"事件后"追踪

季度突发事件（~30% 概率）处理完毕后：

- 伙友跳槽 → 他后来去哪了？不知道。
- 商路被截 → 那条路后来安全了吗？不知道。
- 学徒展露 → 他后来成才了吗？不知道。

事件没有后续、没有追踪、没有"蝴蝶效应"。每一个事件都是孤立的。

---

## 四、终局体验：没有高潮的结束

### 4.1 结局触发突兀

```
上一刻：例行季度结算
下一刻：恭喜！你是守成之主！游戏结束。
```

- 没有终局前兆（"库银已逼近金玉满堂的门槛..."）
- 没有"最后一季"的仪式感
- 玩家可能不记得自己达到了什么条件

### 4.2 结局后缺乏复盘

GameOverPanel 显示"历时 X 年 / 银两 / 信誉"，但缺少：

| 缺失 | 玩家想知道 |
|------|-----------|
| 关键决策回顾 | "我在嘉庆十五年选了赈灾还是自救？" |
| 经营走势图 | "我的银两是怎么一步步涨上来的？" |
| 伙友去向 | "陈守义后来去哪了？" |
| 与历史的对比 | "我比真实晋商如何？" |
| 分支可能性 | "如果当时选了另一个选项..." |

## 五、复玩价值：一遍通关，再无动力

### 5.1 缺乏变化维度

- 没有可选经营路线（激进扩张 vs 保守稳健 — 没有机械层面的分化）
- 没有随机地图/初始条件变化（总是 1 总号 + 2 分号）
- 没有可解锁内容（通关后不开放新内容）
- 不同难度只改变初始数值，不改变核心体验
- 5 个主线事件触发条件和顺序固定

### 5.2 没有"再来一局"的冲动

优秀策略游戏让人立刻想"这次换个打法"。但《晋·信》重开一局：

- 看到的是同样的故事
- 面对的是同样的决策
- 追求的是同样的结局
- 唯一的变量是随机数

## 六、问题严重性矩阵

```
高影响 ──┬── 循环重复无进化 ─ 无建设维度
         │   [S3后流失]        [没有"经营"感]
         │
         ├── 无中期目标 ─── 决策疲劳 ─── 故事割裂
         │   [5年空白]       [120+选择]    [事件无因果]
         │
         ├── 结局突兀 ─── 人脉无用 ─── 简报障碍
         │   [没有高潮]     [三个僵尸数字]  [信息→烦扰]
         │
低影响 ──┴── 无复盘 ─── 复玩零
             [结束后空白]  [无重开动力]
          低复杂度 ──────────────────→ 高复杂度
```

---

## 七、一句话总结

> 游戏有一个扎实的"骨骼"——准确的经济模型、沉浸的历史设定、典雅的 UI 风格。但"血肉"严重不足：玩家在 **第 3 季已见全部内容**，此后 17 季是重复劳动；经营 5 年却感受不到自己在"建设"任何东西；故事演出与机械循环各自为政；结局在某次寻常结算中突然降临，没有高潮、没有复盘。**它有成为优秀游戏的底子，但缺少让玩家"再来一季"的灵魂。**
