## 一、项目概述

《晋·信》是一款以**嘉庆年间晋商票号经营**为背景的文字策略 Web 游戏。玩家扮演票号大掌柜，在白银、信誉、人脉三大核心资源间周旋，经历晨间简报→季度决策→经济模拟→事件剧情→年终合账的完整经营循环，最终达成 6 种结局之一。

**线上地址**：[https://jinxin-twlhpwzz.edgeone.cool](https://jinxin-twlhpwzz.edgeone.cool)  
**GitHub**：[github.com/szm20060312/jinxin](https://github.com/szm20060312/jinxin)

## 三、技术亮点

1. **纯函数引擎架构**：`engine/` 下 2,968 行全部无副作用，与 UI 完全解耦
2. **零外部音频文件**：Web Audio API 实时合成古筝 BGM，免版权
3. **IOA 三层安全**：存档哈希防篡改 + 云端强密码哈希 + 操作行为异常检测
4. **极致轻量**：仅 2 个运行时依赖，gzip 106KB
5. **GitHub 联动部署**：推送即自动上线，EdgeOne 全球 CDN 分发

## 五、已知限制与未来规划

**当前限制**：
- 仅适配桌面端（最小宽度 960px）
- 存档仅限本地 localStorage，不跨设备
- 事件池和剧本总量有限，长期经营后可能出现重复

**未来规划**：
- 移动端触屏适配
- 存档云端同步
- 排行榜与赛季挑战模式
- 社区 Mod 剧本热加载
- 更多历史变局事件（灾荒、战争、商路变化）
